# Interactive Sales Analytics Dashboard

## Overview
This repository contains an interactive **Tableau dashboard** developed to analyze
multi-dimensional sales and customer data. The dashboard is designed to support
marketing and business stakeholders by uncovering trends, patterns, and insights
that inform data-driven sales strategies.
